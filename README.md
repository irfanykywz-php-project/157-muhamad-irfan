# Setup

- composer (Composer version 2.7.6 )
- nodejs (v20.12.2)
- npm (10.5.0)

# How to Run
1. create .env file from .env.example
   - open .env
   - set database configuration in ``DB_CONNECTION``
2. Open project in terminal
3. Run command
   - npm install
   - npm run build
   - composer install
   - php artisan migrate
   - php artisan db:seed
   - php artisan key:generate
   - php artisan serve
4. Open http://localhost:8000
